## Instructions

### About the game
This is an arcade game. To win the game you need to successfully reach the water without colliding with bugs.

### How to install
To start the game just open the index.html file.

### How to play
You can use arrows from your keyboard to move up, down, left and right.
If you touch bugs you lose the game and restart from the beginning point.
Your goal is to reach the water without touching the bugs.
When you reach water you win the game: once you have won, you can start a new game by closing the modal or pressing th "Start a new game" button.
